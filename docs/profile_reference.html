Quick Decision Guide

If your customer / situation is… Use this profile Why
Starting fresh, no specific compliance requirement Security Best Practices Gets everything — full picture, all frameworks shown for reference
Federal agency or bidding on a federal contract FedRAMP NIST 800-53 citations auditors expect to see
DoD contractor, works with CUI data CMMC 2.0 Required for DFARS compliance and contract eligibility
Bank, credit union, investment firm Financial Services SR 11-7 and SR 26-2 are what bank examiners reference
Small business, non-technical owner SMB Plain language, only actionable items, no compliance jargon
Drug company, CRO, medical device (non-clinical) Biotech 21 CFR Part 11, GxP, ICH — what FDA inspectors look for
Hospital, EHR system, clinical AI, telehealth Healthcare HIPAA, HITECH, FDA SaMD — focused on patient data and clinical risk
Deploying AI agents, using MCP servers, LLM tool calls OWASP Agentic Covers the specific attack surface agents introduce — this market is wide open
Selling into EU or have EU customers EU AI Act Enforcement begins August 2026 — customers need this now to prepare
Large enterprise, multiple compliance requirements Run multiple profiles together Sentinel supports stacking — e.g., FedRAMP + OWASP Agentic for a federal AI agent deployment
Note on AI-TOOL checks: The AI-TOOL category (MCP server security, tool call authentication, plugin hijacking) is included in Security Best Practices and OWASP Agentic profiles. Other profiles don't include AI-TOOL checks yet because the category was added after those frameworks were finalized — we can add it selectively as customer demand warrants.

Stacking profiles: You can run multiple profiles in a single scan. The report shows all framework citations side by side. Common combos: FedRAMP + OWASP Agentic (federal AI agents), Healthcare + EU AI Act (EU health systems), Financial + CMMC (defense bank contractors).
