M.A.R.K. — Local website mockup

This directory contains a minimal static mockup for M.A.R.K. (Machine Assisted Real-Time Knowledge).

To preview locally:

1. cd into this directory:
   cd /Users/neepai/hash/workspace/projects/ai-stig-audit/website

2. Start a simple static server. Examples:
   - Python 3 builtin: python3 -m http.server 8000
   - If you have Node: npx serve -s .  (install 'serve' globally for convenience)

3. Open http://127.0.0.1:8000 in your browser.
