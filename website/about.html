About M.A.R.K.

M.A.R.K. stands for Machine Assisted Real-Time Knowledge. This project helps teams capture operational findings, surface recommendations, and keep knowledge accessible and actionable.

Current status (mockup)

  • Prototype agents for lightweight reporting
  • Local storage and simple web UI
  • Security and audit controls in development

Goals

Make operational knowledge searchable, machine-augmented, and tightly integrated with security and compliance workflows.

Sentinel — what it is

Sentinel is a lightweight agent-server system that collects scan reports and operational findings from deployed hosts, stores them for audit and analysis, and surfaces actionable recommendations. Key points:

  • Small-footprint agents report JSON scan results to a central server.
  • Reports are stored in a searchable backend and can be exported in machine-readable formats (SARIF, JSON) for CI/CD and security platforms.
  • Designed for auditability and integration with compliance workflows (CIS/DISA/NIST mappings available).
  • Supports command queueing for remote-triggered scans and has flexible storage adapters (SQLite for local/dev, PostgreSQL for production).
