Machine Assisted Real-time Knowledge

Built to close the gap between AI and security.

M.A.R.K. is a two-product platform — a security audit suite for infrastructure and AI systems, and a self-hosted AI agent runtime that audits itself.

Sentinel

Security audits should be automatic, not manual.

Most teams learn about misconfigurations from penetration tests or — worse — incidents. Sentinel runs the same checks continuously, mapped to the compliance frameworks your team is already using.

Infrastructure coverage

Redis, PostgreSQL, Elasticsearch, Kafka, and MongoDB. Each tool runs fully offline, requires no agent, and works against Docker containers, Kubernetes pods, or direct connections.

AI security

mark-sentinel extends the same approach to AI runtimes — model I/O controls, agent behavior constraints, supply chain checks, and runtime anomaly detection via STIG-style controls.

Compliance-ready output

Every finding carries CIS, NIST, FedRAMP, DISA STIG, CMMC, and MITRE ATT&CK identifiers. SARIF 2.1.0, evidence bundles, and Wiz/SCC JSON come out of every audit run.

Hash

An AI that understands its own security posture.

Hash is a self-hosted personal AI agent runtime. It connects to any model provider, reasons with a multi-pass critique pipeline, builds a persistent knowledge graph from your workspace, and runs Sentinel against itself at startup.

🧠

Reasoning that goes deeper

A three-pass pipeline — draft, critique, synthesize — activates automatically on complex tasks. Uses a mid-tier critic and a frontier model for final synthesis.

🔌

Connects to everything

MCP (Model Context Protocol) integration gives Hash access to any MCP-compatible server. Tools auto-discovered and available to all agents and the interactive REPL.

🧬

Memory that persists

Semantic search via ChromaDB, full-text index, and a live knowledge graph extracted from your workspace. Everything is local — nothing leaves your machine.

🤖

Agents that plan and execute

Give Hash a goal. It decomposes it into a subtask graph, executes each step with checkpoints, and exposes the full lifecycle through a REST API.

🛡️

Self-auditing runtime

Sentinel checks run at startup. A behavioral anomaly detector builds a baseline and flags unusual activity patterns in real time.

📦

Production-ready packaging

Multi-arch Docker image, Compose stack with Ollama, GHCR CI workflow, and a pip install . CLI entry point. Deploy locally or in a container in minutes.

Hash is currently in private development ahead of a public release. Details will be announced once IP filing is complete.
